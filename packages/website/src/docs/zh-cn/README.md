# Mass Block Twitter

Mass Block Twitter 是一款强大的 Chrome 扩展，专门用于解决 Twitter 上的垃圾内容和机器人账号问题。

## 🚀 快速入门

1. [安装扩展](/docs/zh-cn/01-installation)
2. [开始使用](/docs/zh-cn/02-usage)

## ✨ 核心功能

- 自动识别和过滤可能的垃圾/机器人账号
- 基于关键词的内容过滤
- 创建和订阅社区审核列表
- 一键屏蔽功能
- 批量屏蔽/取消屏蔽功能
- 详细的操作仪表盘和数据分析
